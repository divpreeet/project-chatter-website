project-chatter

your personal AI voice assistant

A Python-based voice assistant that uses local models for speech recognition and text-to-speech. Simple to use, easy to customize, and respects your privacy.

Get Started

Features

Pure Python

Built entirely in Python - from the UI to the voice recognition system.

Customizable

Change the wake words, colors, and appearance through simple configuration files.

Privacy-Focused

Doesn't store any data from your voice or computer - no shady stuff under the hood.

About

chatter is an AI voice assistant made purely in Python. It uses the HackClub AI for responses and local models for text-to-speech and speech recognition. Just say "chatter" followed by your message, and it will respond. The project is MIT licensed and designed with simplicity and privacy in mind.

Get Started

  1. Clone the repo: git clone https://github.com/divpreeet/project-chatter.git
  2. Install dependencies: pip install -r requirements.txt
  3. Run chatter: cd main && python3 chatter.py
